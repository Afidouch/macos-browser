DuckDuckGo Privacy Browser commands help

:d :s :g [query]

Search for [query] on the Web

:b [query]

Search for [query] in the Bookmarks

:c [query]

Search for [query] in the Current Window Tabs

:o [query]

Search for [query] among the Tabs in Non-Current Windows

:f :t [query]

Search for [query] among all the Tabs

:x [query]

Fulltext-Search for [query] among all the open Web Pages

:q :a [query]

Search for [query] on DuckDuckGo Instanant Answers

:cp :gg

Copy the current URL to the Clipboard

:dev :ins :src

Show Page Inspector

:help :h
